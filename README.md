# SynkBoard – Professional Creative Suite

A modern, offline-first, **zero-dependency** digital whiteboard / creative canvas built entirely with HTML, CSS, and vanilla JavaScript.

Designed for fast sketching, wireframing, note-taking, UI ideation, or light infographic creation — with a clean, professional interface and smooth drawing experience.

## Key Features & Metrics

- **Fully offline & lightweight** — Works without any internet connection after first load  
- **Zero external dependencies** — No libraries, frameworks, or build tools (except Google Fonts)  
- **Real-time drawing tools**  
  - Brush / Pen (smooth pressure-insensitive strokes)  
  - Eraser  
  - Line, Rectangle, Circle, Triangle shape tools  
  - Text tool with live editable overlay  
- **Undo / Redo** — Full history stack with state snapshots  
- **Color palette** — Instant color switching with visual swatches  
- **Brush size control** — Smooth slider (2–25 px)  
- **Dark / Light theme toggle** — Elegant system-aware design with glass-like elements  
- **Responsive layout** — Mobile-friendly toolbar repositioning  
- **High-performance canvas** — Uses native HTML5 Canvas API for fluid drawing  
- **Professional aesthetic** — Glassmorphism, subtle glows, micro-animations, modern typography (Inter)

## How It Works

1. Open `index.html` in any modern browser  
2. Select a tool from the left toolbar (brush is default)  
3. Choose color from bottom swatches or adjust size with the slider  
4. Draw freely on the infinite canvas  
5. Use text tool → click anywhere → type → press Enter  
6. Shapes: click + drag to draw perfect lines/rects/circles/triangles  
7. Undo/Redo with buttons or clear the board entirely  
8. Toggle dark/light mode anytime

## Technical Highlights

- **Pure vanilla JavaScript** — Event delegation, custom history system, shape preview logic  
- **Smooth drawing experience** — Mouse + touch support (mobile/tablet friendly)  
- **State management** — Canvas snapshot history for undo/redo  
- **Theme engine** — CSS custom properties + data-theme attribute switching  
- **No build step** — Just open the file — ideal for quick demos & sharing  
- **Performance optimized** — Minimal DOM updates, efficient redraw logic  

## Who Is This For?

- Designers & developers doing quick wireframes / moodboards  
- Students & educators for visual note-taking  
- Anyone who wants a beautiful, private, offline drawing tool  
- Portfolio showcase for vanilla JS + Canvas mastery
